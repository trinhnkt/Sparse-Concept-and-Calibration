\begin{abstract}
Knowledge Tracing (KT) models are commonly evaluated using aggregate predictive metrics such as AUC and accuracy. However, overall metrics may hide model behavior on sparse or limited cold‑start Knowledge Components (KCs), where reliable probability estimates are crucial for educational decision support. This paper presents a reproducible diagnostic protocol for sparse‑concept and calibration evaluation in KT. The protocol combines learner‑based and temporal splits, train‑only KC‑frequency stratification, limited cold‑start analysis, calibration metrics (ECE, Brier decomposition), reliability diagrams per stratum, and a seven‑channel leakage‑control audit checklist. We instantiate the protocol on ASSISTments 2012, Junyi Academy, and XES3G5M with baselines BKT, DKT, and SimpleKT. Under our experimental conditions, calibration profiles differ across KC‑frequency strata, and overall AUC rankings can diverge from sparse‑KC calibration rankings. We prepare scripts and report templates for release upon acceptance to support reproducible sparse‑concept and calibration diagnostics for future KT studies.
\end{abstract}

\section*{Artifact Availability \& Reproducibility Checklist}
The anonymized artifact repository will be provided during peer review. The public repository will be released upon acceptance. Consistent with the design principles of Section~\ref{sec:protocol}, the artifact includes the following standardized components:
\begin{itemize}
  \item \textbf{Dataset Preprocessing:} Scripts that normalize raw data into the schema defined in Section~\ref{sec:protocol}.
  \item \textbf{Split Construction:} Reproducible partition logs and multi‑seed split constructors.
  \item \textbf{Prediction‑level Exports:} CSV files containing baseline predicted probabilities and ground‑truth labels.
  \item \textbf{KC‑Strata Assignment:} Code mapping training‑fold frequencies to strata buckets.
  \item \textbf{Calibration Verification:} Scripts computing Expected Calibration Error (ECE).
  \item \textbf{Brier Decomposition:} Implementation of Uncertainty, Reliability, and Resolution equations.
  \item \textbf{Reliability Diagrams:} Plotting utilities for multi‑strata calibration curves.
  \item \textbf{Leakage Audit:} Automated logs verifying channels L1--L7.
  \item \textbf{LaTeX Table Exporters:} Python exporters generating book‑tabs tables.
  \item \textbf{One‑Command Reproduction:} Unified script ensuring machine‑independent reproducibility.
\end{itemize}
